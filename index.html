Checkers Engine

This application implements the Negamax algorithm with alpha-beta pruning and iterative deepening, achieving up to a 14-ply depth search. It uses a transposition table to reduce the search space of its search tree and an efficient 10 x 10 "mailbox" board representation to speed up search times. It also uses quiescence search to avoid the so called "horizon effect". Its heuristic evaluation function accounts for piece value, piece position, and win conditions.

Connect Four Engine

This application implements the Minimax algorithm with alpha-beta pruning, achieving up to a 8-ply depth search in a reasonable timespan. It uses an immutable 6 x 7 "mailbox" board representation for programming convenience. Its heuristic evaluation function values moves that lead to future 3-in-a-row and 4-in-a-row combinations.

Handwritten Digit Classifier

This is a handwritten digit classifier trained on the popular MNIST dataset, achieving a 94.89% accuracy on the MNIST validation set. It uses a self-coded two-layer, fully-connected neural network implementation (784 x 74 x 10) with a sigmoid activation function. Each layer of the network is represented as a matrix of weights, and feedforward and backpropagation operations are represented as a series of matrix multiplications and dot products.

Harmless x86 computer virus for Windows OS View on GitHub

x86 Assembly Virus

This is a harmless computer virus that infects x86 PE (portable executable) files on Windows operating systems. It uses a directory-crawling algorithm to find and infect other .exe files on the host's file system. The virus works by first appending itself to the end of a .exe file. From there, it manipulates the file's PE header, changing the entry-point of the program to point to the start of the appended virus code. It then alters various PE header fields to avoid detection by simple anti-virus scans.

WebGL Model Drawer

This is a WebGL application to draw and render cylindrical splines in browser. It features an interface to filter between different shading techniques, including Gouraud, Phong, depth, rim, and cel (toon) shading. It also allows users to alter the ambient and diffuse light colors as well as the specular constant for the scene. The application implements alpha selection to allow users to interact with objects in the scene and move them around using the mouse.
